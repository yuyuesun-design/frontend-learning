# 前端基础学习项目 (frontend-learning)

> 一名设计师从 0 开始学前端的练习项目，记录学习过程中的第一个完整网页。

## 项目简介

这是我学习前端三件套（HTML / CSS / JavaScript）的第一个练习项目。
通过搭建一个个人主页，我把入门阶段最常用的标签和概念都实践了一遍。

## 用到的技术

- **HTML**：负责页面结构（标题、段落、列表、表格、表单、图片、分区等）
- **CSS**：负责页面样式（背景色、边框、圆角、间距、卡片样式等）
- **JavaScript**：负责页面交互（点击按钮显示文字、切换卡片样式等）
